리눅스 명령어
- echo <"출력할문자열"> [>> 파일명] : 콘솔 출력. 파일명을 뒤에 쓰면 그 파일에 문자열을 이어 쓴다.
- rm <파일명> : 파일 삭제
- history : 이전에 썼던 명령어 모음집

깃 명령어 : 앞에 꼭 git 붙이기
- branch -M main : 현재 branch의 이름을 main으로 변경한다.(pride의 관례상 그래왔음)
- remote add origin https://github.com/아이디/원격리포지토리이름.git : 로컬 저장소를 원격 저장소와 연동한다. (push해도 히스토리가 꼬이진 않음. git이 막아줌)
- push -u origin main : origin(원격)에 main 브랜치에 푸시하고, 다음에 "push"만 써도 푸시되게 한다.
- push : 원격 저장소(현재 origin)에 업데이트된 커밋을 올린다.
- remote -v : 현재 origin(원격)이 무엇인지 확인한다.
- remote remove origin : 다른 사람의 저장소를 내 저장소에 올리고 싶을 때 같은 때 origin을 바꿔야 하는데 쓴다. 이거 쓴 후에 remote add origin ~~하면 된다.
- clone https://github.com/아이디/원격리포지토리이름.git : 원격 저장소를 로컬로 가져온다.
- clone https://github.com/아이디/원격리포지토리이름.git . : "현재 폴더에" 원격 저장소를 로컬로 가져온다.
