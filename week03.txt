리눅스 명령어
- ls : 현재 디렉토리의 파일/폴더 목록. "-a"를 뒤에 쓰면 숨김 폴더도 표시된다.
- touch <파일명> : 그 뒤의 이름을 가진 빈 파일 생성
- cd <경로> : 그 경로로 작업 폴더를 이동함. ".."를 쓰면 상위 폴더, "."를 쓰면 현재 폴더로 이동됨
- cat <파일명> : 그 뒤의 이름을 가진 파일의 내용 출력

깃 명령어 : 앞에 꼭 git 붙이기
- init : 현 디렉토리를 git 디렉토리로 지정
- status : 현재 git 디렉토리의 상태 표시.
- add <파일명> : 그 뒤의 이름을 가진 파일을 스테이징. "."을 쓰면 현재 폴더의 모든 파일을 스테이징함.
- config --global user.email "개인이메일주소" : commit하는 사용자의 이메일 주소 설정
- config --global user.name "ID" : commit하는 사용자의 ID 설정
- config --list : 설정값 확인
- commit -m "커밋 메시지" : 스테이징한 파일을 온라인에 올림.
- log : 커밋 기록 확인
- checkout <해시코드 7자리 또는 전체 또는 "-"> : 특정 커밋 시점으로 돌아감. "-"를 쓰면 바로 이전 커밋 시점으로 돌아감.
